Inventory Manager (C-Language Practice) 📦
Этот проект представляет собой консольный менеджер товаров, написанный на языке C. Программа была создана в учебных целях для изучения низкоуровневой логики и понимания ключевых различий между C и C++ / Java.

🎯 Цель проекта
Основная задача — реализовать типичный функционал CRUD (Create, Read, Delete), используя только стандартную библиотеку C (stdio.h, stdlib.h), без использования классов, векторов или строк из C++.

Что этот проект помогает понять:
Память и Структуры: Вместо классов используются struct. Здесь нет инкапсуляции — все данные открыты.

Управление строками: Работа с массивами символов (char[]) вместо объекта std::string.

Процедурный подход: Логика строится на функциях и передаче данных, а не на методах объектов.

Статическая память: Использование массивов с фиксированным размером (#define MAX_PRODUCTS 100).

🛠 Функционал
Генерация ID: Автоматическое создание уникального трехзначного идентификатора для каждого товара.

Управление инвентарем: Возможность добавлять новые товары (имя, цена) и удалять их по ID.

Навигация: Реализовано интерактивное консольное меню с защитой от неверного ввода.

Просмотр: Динамическое отображение текущего списка товаров.
